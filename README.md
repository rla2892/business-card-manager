# business-card-manager
Typescript practice

## Prerequisites

- Node.js version 20.17.0

## Build and Start

To build and start the project, follow these steps:

1. **Install dependencies**:
	```sh
	npm install
	```

2. **Build the project**:
	```sh
	npm run build
	```

3. **Start the project**:
	```sh
	npm run build-and-start
	```
